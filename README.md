# SystemProgram
## Install
What things you need to install the software and how to install them
```
1. Open terminal in folder
2. run 'make clean'
3. run 'make'
```

## Usage
Follow these steps:
```
1. su
2. insmod printerDriver.ko
3. sudo ./testPrinterDriver
```

To view logfile, follow:
```
1. dmesg | tail -20
```
